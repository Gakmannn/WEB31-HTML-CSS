Скачать и установить Git

Перезагрузить VSCode

Перейти в книгу, пункт 1.6 https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup

Ввести 2 команды в VSCode (Установка имени и почты git)

git config --global user.name "John Doe"

git config --global user.email johndoe@example.com

Нажать на шестерёнку, включить синхронизацию через GitHub
